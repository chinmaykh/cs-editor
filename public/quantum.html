Continuous Random Variables

Here's an overview about the a special class of random variables continuous ones .

An example, the exact moment when a certain event will happen. You can't say when exactly something will happen. You'll define more of a region in time.

For such situation, we idenitfy that we don't have the probability defined for a particular instance. It will be 0. Instead, we shall have to look at the probability over a certain region.

Probability distribution function

It is exactly as it sounds. It's a function that show's how the spread of the probability is.

Probability of an event corresponds the probability of the random variable being in a some range of possible values. Integration of the distribution function in the given region provides the required probability.
